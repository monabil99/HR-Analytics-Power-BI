# HR Analytics Dashboard | Atlas Labs

## Project Overview

This project is an interactive HR Analytics Dashboard developed in Power BI to analyze workforce demographics, employee performance, hiring trends, and attrition metrics.

The dashboard enables HR teams and decision-makers to monitor employee-related KPIs and gain actionable insights through interactive visualizations.

## Dashboard Pages

### 1. Overview

* Total Employees
* Active Employees
* Inactive Employees
* Attrition Rate
* Employee Hiring Trends
* Department Analysis

### 2. Demographics

* Employee distribution by age groups
* Education levels
* Gender analysis
* Workforce composition

### 3. Performance Tracker

* Employee performance reviews
* Manager ratings
* Job satisfaction trends
* Work-life balance analysis
* Performance history tracking

### 4. Attrition Analysis

* Attrition rate by hire date
* Attrition by business travel
* Employee turnover insights
* Retention metrics

## Tools & Technologies

* Power BI
* Power Query
* DAX
* Data Modeling
* Interactive Visualizations

## Key Insights

* The majority of employees are concentrated within a limited number of departments.
* Attrition is not evenly distributed across employee groups.
* Business travel appears to influence employee turnover patterns.
* Employee satisfaction and manager ratings provide valuable indicators of workforce engagement.
* Hiring trends fluctuate across different years and departments.

## Recommendations

* Investigate departments with elevated attrition rates.
* Monitor employee satisfaction and work-life balance metrics regularly.
* Review business travel policies if turnover is significantly higher among frequent travelers.
* Implement retention programs targeting high-risk employee segments.
* Use performance and satisfaction indicators proactively to improve employee engagement.
